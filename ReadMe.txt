C++

Да се дефинира базов клас за студент зададен с име, факултетен номер, номер на група и курс. 
Да се дефинира производен клас за успех зададен с динамичен масив от оценките по всички дисциплини 
до момента (дължината на масива зависи от курса – счита се, че за семестър броят на оценките е 5). 
Класовете да включват конструктори без параметри за попълване на член-променливите в диалог. Да се 
определят и извеждат средният успех за всеки студент и максималния среден успех за зададена група. 
Да се извежда пълната информация за всеки студент и за студент, зададен с факултетния си номер. 
Да се предефинира операцията <<, която да се използва за извеждане на данните. 
Данните да се съхраняват във файл.

To define a base class for a student set with name, faculty number, group number and course. 
To define a derived class for success set with a dynamic array of the grades in all disciplines to date 
(the length of the array depends on the course - the number of grades is considered to be 5 per semester). 
Classes should include parameterless constructors for populating member variables in a dialog. To determine 
and output the GPA for each student and the maximum GPA for a given group. To display the complete information 
for each student and for a student assigned with his faculty number. Redefine the << operation to be used to output the data. 
Data to be stored in a file.
